# Krupalu Dashboard 
 
A business management dashboard built with Flask and SQLite for managing fabric printing orders, deliveries, and payments. 
 
## Features 
- Order management and tracking 
- Delivery scheduling and confirmation 
- Payment and balance tracking 
- Reports with charts and KPI cards 
- Party (customer) management 
 
## Tech Stack 
- Python (Flask) 
- SQLite 
- HTML/CSS/JavaScript 
- Chart.js for reports
